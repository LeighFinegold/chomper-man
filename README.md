# chomper-man
AI generated game like flappy birds but takes an image of a friend/family member

## Play Online
The game is automatically deployed to GitHub Pages and is accessible at:
[https://leighfinegold.github.io/chomper-man/](https://leighfinegold.github.io/chomper-man/)

## Deployment
A GitHub Actions workflow (`.github/workflows/deploy.yml`) automatically publishes the latest `index.html` to GitHub Pages whenever a pull request is merged into `main`. No manual steps are required â€” after every merge the live site is updated within minutes.
